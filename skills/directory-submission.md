# Directory Submission Skill

## What this skill does

This skill automates submitting salestools.club to AI/SaaS directories for backlink building. It uses Browser MCP to navigate to submission pages and fill out forms.

## When to use this skill

- When you need to submit salestools.club to directory websites
- When building backlinks for SEO
- When expanding the site's presence in AI tool ecosystems

## Site being submitted

- **Name:** Salestools Club
- **URL:** https://salestools.club
- **Description:** The sales API directory for AI agents. Find MCP servers, CRM APIs, and agentic sales tools for Claude Code, Gemini CLI, and autonomous sales workflows.
- **Category:** Business > Sales > AI Tools
- **Tags:** sales API, MCP server, CRM, AI agents, sales automation, lead enrichment

## How to use with Browser MCP

1. Start Browser MCP server: `npx @browsermcp/mcp`
2. Navigate to each directory's submit page
3. Fill in the form with site info:
   - Name: "Salestools Club"
   - URL: "https://salestools.club"
   - Description: Use the short description from above
   - Category: Select relevant category
4. Submit the form
5. Move to next directory

## Notes

- Skip product launch platforms (Product Hunt, etc.)
- Most directories are free
- Some may require email verification
- Track submissions in a spreadsheet for follow-ups
