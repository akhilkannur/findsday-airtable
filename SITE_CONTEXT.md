# Site Context Reference

## Site: Salestools.club

**What it is:** A directory for AI-Native Operators (founders, sales managers, revenue ops) to find sales APIs, MCP servers, and agentic sales tools.

**Target Audience:** Non-developers who use AI (Claude, Cursor, Gemini CLI) to build sales workflows.

**Key Features:**
- Lists sales tools/APIs with MCP server support
- Provides skill files for AI agents
- Categories: CRM, lead enrichment, outreach, meeting scheduling, etc.
- SEO-focused directory for sales tech

**Value Prop:** "Lego blocks for sales" - specialized APIs that AI agents can plug into.

**Submit Page:** https://salestools.club/submit

**Ideal Directory Categories:** AI tools directories, SaaS directories, Marketing/SEO directories, Developer tool directories, Business tool directories.
